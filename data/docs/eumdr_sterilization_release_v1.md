# Work Instruction Sterilization Load Release

## Purpose
Define evidence needed before sterilized load release.

## Controlled Requirements
1. A biological indicator challenge shall be run at least once per week for each validated sterilizer.
2. The sterilization batch packet shall include the signed load configuration record before release.
3. A manufacturing engineer or a designated quality engineer may release a sterilization load.
